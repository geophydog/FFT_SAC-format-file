# FFT_SAC-format-file
- To execute FFT of a SAC format file and virtualize the result;
- Run on Linux or Mac OS platform.
***

# Usage
```
fft_out sacfile fre_high_limit output_result    
```
***
# Denpendencies
- GMT (the Generic Mapping Tools), here version of 5.3.1
